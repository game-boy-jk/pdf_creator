# PDF Creator

FastAPI service that fills text placeholders in PDF templates and stores generated PDFs in S3/MinIO.

Templates are regular PDFs exported from Word/LibreOffice/etc. The service does not create templates in code. Put labels directly in the document text:

```text
Contract with {{customer_name}}
Date: {{date}}
Total: {{total_sum}}
```

## API

`POST /generate`

```json
{
  "template_id": "templates/contract.pdf",
  "data": {
    "customer_name": "OOO Romashka",
    "date": "2026-05-20",
    "total_sum": "12 500.00 RUB"
  }
}
```

Response:

```json
{
  "file_id": "generated/<uuid>.pdf",
  "file_url": "http://localhost:9000/pdf-files/generated/<uuid>.pdf"
}
```

`file_id` is generated by the service.

## Configuration

Required:

```env
MINIO_ENDPOINT=minio:9000
MINIO_PUBLIC_ENDPOINT=localhost:9000
MINIO_ACCESS_KEY=minioadmin
MINIO_SECRET_KEY=minioadmin
MINIO_BUCKET=pdf-files
```

Optional:

```env
MINIO_SECURE=false
OUTPUT_PREFIX=generated/
CACHE_TTL_SEC=300
PDF_FONT_OBJECT_KEY=fonts/arial.ttf
CONFIG_SERVER_URL=http://config-server:8888
CONFIG_APP_NAME=pdf-creator
CONFIG_PROFILE=default
CONFIG_LABEL=main
CONFIG_FAIL_FAST=false
```

`PDF_FONT_OBJECT_KEY` should point to a TTF/OTF file in the same bucket when templates use subset fonts or Cyrillic text.

## Run

```bash
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

or:

```bash
docker compose up --build
```

## Test
Local tests:
```bash
pytest -q
```
